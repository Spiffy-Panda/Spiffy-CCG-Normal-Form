# Conduit — Archived

This folder holds an earlier iteration of this project's card-game design, internally named **Conduit**. Conduit has been superseded by **Resonance** (see `design/GameRules.md` and `encoding/`) and is kept here for historical reference only.

## Contents

- `RULES.md` — Conduit's rules document. Introduces the seven-faction system that Resonance eventually narrowed to five, and a different combat model.
- `FACTIONS.md` — The faction rework that pushed Conduit to seven factions. Contains notes on Flock and Drift, two factions that did not carry into Resonance.
- `CARDS.md` — Sample Conduit cards. Uses Conduit terminology (Tactics, Relics, Ward, Siphon, etc.), not Resonance terminology.
- `7thColorChatLog.md` — A working chat log about whether a seventh faction was warranted. Design-process artifact.

## Do not cross-reference

Do not treat Conduit mechanics as valid in Resonance contexts. Terminology that differs:

| Conduit | Resonance |
|---|---|
| Tactic | Maneuver |
| Relic | Standard |
| Discard | Cache |
| Void | Cache (exile is a separate state) |
| Ward | Fortification (via Ramparts) |
| Swift | Blitz |
| Life / Energy | Conduit Integrity / Aether |
| Ember / Pyre / Loam / Tide / Shade / Keep / Flock / Drift | EMBER / BULWARK / TIDE / THORN / HOLLOW |

Resonance is **not** a rename of Conduit — the combat model, resource model, and faction system were all redesigned. Treat this archive as a separate-universe reference, not a prior version to port from.
