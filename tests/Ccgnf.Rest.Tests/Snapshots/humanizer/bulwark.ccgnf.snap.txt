=== SealTheBreach ===

=== HoldTheLine ===

=== Reconstitute ===
Resolves: target Conduit where c.owner is you: if not target.arena has collapsed for you, heal starting_integrity_of(target) on target; otherwise SetCounter(target, integrity, 3); SetFlag(Arena[target.arena].collapsed_for[self.controller], false).

=== TheQuietWall ===
Static (KeywordSet(unit)): for each friendly Unit, grant Sentinel to u.
Static (ClashContribution(unit)): when Peak BULWARK, for each friendly Unit, OverrideContribution(u, projected_force: 1).

Static (DamageReduction(Conduit, by_opposing)): when Peak BULWARK, for each Conduit where c.owner is you, Floor(c.integrity, min: 1, source_filter: e -> e.controller != self.controller).
